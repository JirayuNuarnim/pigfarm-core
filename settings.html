ตั้งค่า / ผู้ใช้งาน

ขั้นตอนเพิ่มผู้ใช้ใหม่ (ทำครั้งเดียวต่อคน)

  1. ไปที่ Supabase Dashboard → Authentication → Users → กด "Add user" ใส่อีเมล+รหัสผ่านให้พนักงานคนนั้น
  2. คัดลอก User UID ของบัญชีที่สร้าง (คอลัมน์ UID ในตารางผู้ใช้)
  3. นำ UID มากรอกในฟอร์มด้านล่าง พร้อมกำหนดชื่อ/บทบาท/ฟาร์ม/หมวดที่ดูแล
